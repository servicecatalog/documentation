Werbebanner anpassen

Auf der Seite Werbebanner anpassen können Sie den Bereich des Marktplatzes anpassen, auf dem zum Beispiel Werbung, Ihr Firmenlogo oder weitere Informationen angezeigt werden.

Das Werbebanner kann als Bild zur Verfügung gestellt werden oder als HTML-Code, über den beispielsweise animierte Grafiken eingebunden werden. Sie können für jede unterstützte Sprache je ein eigenes Werbebanner für Desktop- und mobile Geräte hinterlegen. Bevor Sie ein Werbebanner veröffentlichen, können Sie es testen und in einer Vorschau anzeigen lassen. Sobald Sie die Vorschau bestätigen, ist das Werbebanner öffentlich und für alle Besucher des Marktplatzes sichtbar.

Das Branding eines Marktplatzes umfasst auch das Layout (Farben, Schriften, Hintergrundbilder, Logos, usw.) und die verwendeten Texte (für Elemente der Benutzeroberfläche, E-Mails, usw.). Zum Anpassen des Layouts verwenden Sie die Option Layout anpassen im Menü Marktplatz. Die Texte können Sie über die Option Texte anpassen im Menü Marktplatz ändern.

So ändern Sie das Werbebanner eines Marktplatzes:

  1. Wählen Sie in der Liste Marktplatz den Marktplatz, für den Sie das Werbebanner ändern möchten.

    Die Liste stellt alle Marktplätze zur Auswahl, auf die Sie als Marktplatz-Manager Zugriff haben.

  2. Wählen Sie die Sprache, für die die Anpassung erfolgen soll.

    Sie können für jede unterstützte Sprache Werbebanner hinterlegen. Wenn Sie für eine Sprache kein Banner angeben, wird das Banner der als Standard eingestellten Sprache verwendet. Wenn für diese ebenfalls kein Werbebanner angegeben ist, wird das Banner verwendet, das standardmäßig nach der Installation der Plattform zur Verfügung steht.

  3. Geben Sie den HTML-Code ein, der einen Verweis auf das Bild Ihres Werbebanners enthält. Zur optimalen Darstellung können Sie für Desktop-Geräte und mobile Geräte unterschiedliche Bilder verwenden. Das Bild für Desktop-Geräte wird verwendet, wenn die Display-Breite 768 Pixel überschreitet. Bei kleineren Breiten wird das Bild für mobile Geräte benutzt.

    Wie für Werbebanner üblich sollte das Bild für Desktop-Geräte eine Größe von 1600 x 500 Pixel oder ein Vielfaches davon mit demselben Seitenverhältnis haben. Für mobile Geräte sollte die Größe 768 x 940 Pixel oder ein Vielfaches davon mit demselben Seitenverhältnis sein. Empfohlen wird eine Breite von 100% für das Werbebanner. Die Höhe für Desktop-Geräte sollte zwischen 200 und 300 Pixel liegen, für mobile Geräte zwischen 680 und 720 Pixel.

    Sie können den HTML-Code direkt eingeben oder einen eingebetteten Frame verwenden, um das gewünschte Werbebanner einzubinden.

    Beispiel: <IFRAME FRAMEBORDER="0" width="100%" height="290px" marginwidth="0" marginheight="0" SRC="IhrWerbebanner.htm" SCROLLING="no"></IFRAME>

    Hinweis: Es wird empfohlen, einen eingebetteten Frame zu verwenden. Wenn Sie keinen eingebetteten Frame verwenden, sollten Sie nur Snippets als HTML-Code angeben. Verwenden Sie keine vollständigen HTML-Seiten, die Elemente wie <html>, <head> oder <body> enthalten. Überprüfen Sie das Layout der Seite mit Vorschau (siehe nächster Schritt). Durch ein Snippet kann eventuell die Startseite des Marktplatzes, die Vorschau oder auch die Seite Werbebanner anpassen nicht mehr korrekt angezeigt werden, vor allem dann, wenn das Snippet Script enthält (z.B. JavaScript).

    Beachten Sie, dass Sie für eine fehlerfreie Seitendarstellung verantwortlich sind.

  4. Testen Sie ein geändertes Werbebanner mit Vorschau.
  5. Sobald das neue Banner Ihren Vorstellungen entspricht, bestätigen Sie die Veröffentlichung der neuen Startseite des Marktplatzes mit Ja. Das neue Werbebanner wird sofort veröffentlicht und ist für die Benutzer beim nächsten Besuch des Marktplatzes sichtbar.

    Mit Nein gelangen Sie wieder zur Seite Werbebanner anpassen und können weitere Änderungen vornehmen. Das existierende Werbebanner wird nicht geändert.

  6. Speichern Sie mit Sichern alle Änderungen, die Sie vorgenommen haben. Geänderte Werbebanner werden veröffentlicht und sind für die Benutzer beim nächsten Besuch des Marktplatzes sichtbar.
